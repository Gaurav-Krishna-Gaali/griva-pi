# Image Comparison Screen - Medical Imaging Tool

## Overview
The Image Comparison Screen is a powerful tool designed specifically for medical imaging applications like colposcopy. It allows healthcare professionals to compare multiple images side-by-side with advanced viewing modes and medical imaging tools.

## üöÄ **Key Features**

### **Three View Modes**
1. **Grid View (2x2)**: Compare up to 4 images in a grid layout
   - First slot always shows the current/primary image
   - Three additional slots for comparison images
   - Perfect for before/after comparisons or multiple angle views

2. **Single View**: Focus on one image with full-screen detail
   - Interactive zoom and pan capabilities
   - Medical grid and ruler overlays
   - Ideal for detailed analysis

3. **Side-by-Side View**: Compare two images directly
   - Left: Current/primary image
   - Right: Comparison image
   - Best for direct A/B comparisons

### **Medical Imaging Tools**
- **Grid Overlay**: Toggle 10x10 grid for precise measurements
- **Ruler Overlay**: Top and left rulers with measurement markings
- **Interactive Zoom**: Pinch to zoom from 0.5x to 4x magnification
- **Pan Navigation**: Drag to move around zoomed images

### **Image Management**
- **Smart Slot System**: First slot is protected (current image)
- **Easy Addition**: Tap empty slots to add images from gallery
- **Quick Removal**: Remove comparison images with one tap
- **Image Selection**: Choose from all available captured images

### **Medical Tagging System**
Pre-configured medical tags for quick categorization:
- Normal, Abnormal, Suspicious
- Benign, Malignant
- Inflammation, Erosion, Polyp, Cyst
- Custom "Other" option

### **Professional Features**
- **Save Comparisons**: Store comparison data for future reference
- **Export Functionality**: Export comparisons for reports
- **Responsive Design**: Works on all screen sizes
- **Medical UI**: Clean, professional interface optimized for medical use
